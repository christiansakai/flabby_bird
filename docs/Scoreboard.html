Class: Scoreboard

Scoreboard

Class representing Scoreboard. Contains score, best score, and additional fancy animations.

Constructor

new Scoreboard(config)

Create a Scoreboard. This group is created by creating a board, score, best score, and emitter animation. An emitter is an object that emits particle-like Physics objects.
