Class: PipePair

PipePair

Class representing a pair of Pipes. A pair of Pipes is a group of normal Pipe with an upside down Pipe.

Constructor

new PipePair(config)

Create a Pair of Pipe. This group is created by creating a normal pipe (bottom pipe) and an upside down pipe (top pipe).
